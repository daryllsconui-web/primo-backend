# Knowledge Base Instructions

Drop your files in this folder. Supported formats:
- .txt — plain text files
- .md  — markdown files  
- .pdf — PDF documents
- .docx — Word documents
- .png / .jpg / .jpeg — images (text is extracted via OCR)

The AI agent will automatically load ALL files in this folder every time the backend starts.

## How to add knowledge:
1. Place your files in this folder (backend/knowledge/)
2. Restart the backend: the files are automatically ingested
3. The AI will now answer questions based on the content

## Examples of what to add:
- Company profile / About us document
- Product or service catalog
- FAQ document
- Price list
- Team bios
- Any reference material you want the AI to know about
